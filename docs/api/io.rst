I/O Module (``modules.io``)
============================

Auto-detect format from extension and load or save any supported data type.

Supported extensions: ``.pkl``, ``.npy``, ``.npz``, ``.csv``, ``.xlsx``, ``.tsv``,
``.json``, ``.parquet``
