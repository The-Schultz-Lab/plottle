Plotting Module (``modules.plotting``)
=======================================

All 13 plot types across Matplotlib, Seaborn, and Plotly.

Return conventions (DEC-003):

* **Matplotlib / Seaborn** functions return ``(fig, ax, info)``
* **Plotly** functions return ``(fig, info)``
